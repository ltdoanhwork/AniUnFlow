\chapter{Conclusion \& Future Work}

\section{Conclusion}

This thesis presented \textbf{AniUnFlow}, a structure-first unsupervised optical-flow framework for 2D animation. The central idea is that animation correspondence should not be treated as a purely dense matching problem. Instead, AniUnFlow combines SAM-guided object structure, temporal slot memory, dense local refinement, propagated structural agreement, and boundary-aware residual correction so that motion estimation remains both coherent and expressive in stylized scenes.

The most important outcome of this work is not a single scalar benchmark value, but a clearer design principle for animation flow estimation. Object structure is valuable because it organizes the motion hypothesis in flat, ambiguous regions. Dense refinement is valuable because it restores local articulation and geometric detail that an object prior alone cannot capture. Propagation memory is valuable because it tells the framework when nearby temporal structure is still trustworthy after motion has already been estimated. Together, these components produce a framework that is easier to interpret, easier to visualize, and more aligned with the practical failure modes of anime-style motion.

Another contribution of the thesis is presentational. By rewriting the work around one shared framework name and adding more schematic and qualitative figures, the paper now reads less like a collection of disconnected experiments and more like a coherent research story. This matters because the development of AniUnFlow was inherently iterative: the architecture improved not only by adding modules, but by learning how structure, memory, and dense refinement should interact.

\section{Limitations and Future Work}

AniUnFlow still leaves several open directions for future work.

\begin{itemize}
\item \textbf{Very large displacement remains difficult.} Even with structure-first reasoning, extremely large motion can exceed what local dense refinement can recover from a conservative prior. The latest large-motion extension suggests that broader search and visibility-aware fusion are promising next steps.
\item \textbf{Propagation memory is still a heuristic structural cue.} The propagated agreement map improves refinement stability, but it remains an indirect signal derived from masks and current motion hypotheses. Stronger learned reliability estimation could make this pathway more adaptive.
\item \textbf{Stylized animation effects remain under-modeled.} Smear frames, impact frames, motion streaks, and intentional shape exaggeration can violate ordinary correspondence assumptions. Handling such phenomena may require animation-aware effect detection or richer temporal priors.
\item \textbf{Downstream evaluation should be expanded.} The framework is naturally relevant to interpolation, editing, and temporally consistent stylization, but these tasks deserve fuller dedicated evaluation beyond the scope of the current thesis.
\end{itemize}

In summary, AniUnFlow provides a practical foundation for animation-oriented optical flow by showing that structure and dense correspondence should cooperate rather than compete. The framework already supports richer visualization and more readable analysis than the earlier draft of the thesis, and it opens a clear path toward future work on larger motion, stronger temporal reasoning, and production-oriented animation tools.
